            const getValues = () => {
              let userString = document.getElementById('user-string').value;
              let revString = reverseString(userString);
              displayString(revString);
            }
            
            const reverseString = (input) => {
              let revString = '';
              for (let i = input.length - 1; i >= 0; i--) {
                let letter = input[i];
                revString += letter;
              }
              return revString;
            
            }
            
            const displayString = (output) => {
              document.getElementById('results').textContent = output;
              let alertBox = document.getElementById('alert');
              alertBox.classList.remove('invisible');
            
            }

The code is structured in three functions

getValues

getValues() is a function that grabs the text from the input element and passes this text as a arguement to the function reverseString()

reverseString()

reverseString() is a function that recieves the text from the method getValues() and reverses the string with a for loop. After the string is revresed, the string is then returned to the displayString() function

displayString()

displayString() is a function that uses the text returned from reverseString() and displays the text in a span tag. This function also removes the invisble class on the div with a id of alert in order for the span tag inside of the alert div to show the reversed string.
